# MarinVocab
MarinVocab is a diagnostic benchmark for maritime open-vocabulary surface-target detection. It provides object-level bounding-box annotations, base/novel category splits, structured diagnostic labels, and evaluation protocols for studying open-vocabulary perception in complex waterborne transportation scenarios.
